Javascript Ignite 2015

Learn from Hong Kong topmost Javascript experts!

Download Download View Javascript Ignite 2015 on GitHub

Javascript Ignite! is gathering the best Javascript developers in Hong Kong and teach eager learners to get hold of Javascript. It converts programmers from other language to a Javascript Adept in a 4 x 3hour workshop.

Target Audience

  • You need to be fluent in typing, we will code during sessions.
  • You will not get faint when seeing code =] Better, you LOVE it =3=
  • We assume you have no experience of programming in Javascript.
  • You are interested in programming in Javascript.

Basic course structure

Session 1: Welcome to the Javascript World

20 min - Self introduction of Gurus + Course introduction
30 min - Javascript 101 and Ecosystem introduction
10 min - Break
50 min - Code Academy Part I - V
10 min - Break
50 min - Code Academy Part I - V
10 min - Break
Deliverable: CodeAcademy related

Session 2: Welcome to the Javascript World II

10 min Warm up
40 min CodeAcademy Part VI - VIII
10 min Break
50 min CodeAcademy Part VI - VIII
10 min Break
50 min CodeAcademy Part VI - VIII
10 min Break
Deliverable: CodeAcademy related

Session 3: Hello Real World

20 min Describe the To-Do List and let learners brainstorm 30 min Design phrase
10 min Break
30 min Discussion
20 min Coding
10 min Break
30 min Coding
20 min Polling of next workshop content
10 min Break
Deliverable: A Todo List with 1 additional feature from learners

Session 4:

Core: git and Pull Request Depends on the interest of learners: Backend & Deployment: NodeJS on Heroku Visualization : D3.js Other? Game Development? Aim to explore together and give them a taste in the industry. Core Deliverable: a github page deployed with the web app they developed

Intended Learning Outcomes (ILO's)

  • boot a web app from scratch
  • deployed to the web
  • share with their friends.
  • Self teach themselves in various sources.
